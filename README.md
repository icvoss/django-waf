# django-icv-waf

Self-hosted request filtering, bot management, and WAF middleware for Django.

Provides rate limiting, user-agent anomaly scoring, JS proof-of-work challenges,
nginx blocklist generation, and a collective threat feed integration — all
configurable without a reverse-proxy vendor.

For full documentation, configuration reference, and API details see the
[APP-018 spec](../../docs/specs/APP-018-waf/).
